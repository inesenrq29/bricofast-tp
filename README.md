# bricofast-tp
Où sont stockées les données ?: dans des variables 

Quelles parties du code sont dupliquées ? : exceptions - messages d’erreurs

Qu’est-ce qui rend le code difficile à maintenir ?: le choix des noms des variables (pas assez explicites), les boucles imbriquées et répétitives, pas de commentaires, l’initialisation des booléens, try catch dans if (plutot if dans try catch ?)

Quels sont les concepts métier évidents non modélisés ? notion de classe
